| **B站合集+百度网盘** | https://www.bilibili.com/list/ml2040282687 (璃月69首示例) | MP3 | 网盘打包下载、提取码公开 |

| **游戏内文件提取** | 游戏安装目录 `GenshinImpact_Data/StreamingAssets/Audio` | WEM(需转换) | 最全最准，但需 WWise 解码工具 |

### 推荐方案

- **开发调试阶段**：从爱给网或 B站网盘合集获取少量代表性曲目（每国度 4 首：黎明/白天/黄昏/夜晚）即可跑通流程
- **正式部署阶段**：建议用户自备音频文件放置于项目 `public/audio/` 目录，配置文件仅记录文件名映射，不随仓库分发版权音频
- **高品质需求**：贴吧 Hi-Res FLAC 合集适合追求音质场景，但文件较大需压缩为 MP3/AAC 供 Web 播放

## 三、瓦片地图下载渠道

这是项目地图渲染的底层基础。调研发现两条可行路径：**本地瓦片方案**（推荐）和**在线图源方案**。

### 3.1 本地瓦片方案 ⭐ 推荐

将瓦片图片下载到项目 `public/images/map/{z}/{x}/{y}.png`，通过 Leaflet 的 `L.tileLayer` 本地加载。

**参考开源项目**：

| 项目 | 链接 | 技术栈 | 价值 |
|------|------|--------|------|
| **shaoyahu/genshin.impact.map** | https://github.com/shaoyahu/genshin.impact.map | Vite+React+Leaflet | 瓦片本地化方案、坐标系配置、区域锚点数据格式，已克隆分析 |
| **genshin-map/genshinmap.github.io** | https://github.com/genshin-map/genshinmap.github.io | React+Leaflet | 基于 yuanshen.site 翻译版，GPLv3 开源 |
| **空荧酒馆 yuanshen.site** | http://www.yuanshen.site/ | — | 中文原版社区地图，瓦片源头 |

**已核实的坐标系配置**（来自开源项目源码）：

```js
// Leaflet 地图初始化配置
L.CRS.Simple                    // 使用简单坐标系（非地理坐标）
maxBounds: [(-77, 56), (-180, 202)]  // 地图边界（左上、右下）
center: [-84, 148]              // 初始中心点
minZoom: 1, maxZoom: 7          // 缩放层级
tileLayer: "/images/map/{z}/{x}/{y}.png"  // 本地瓦片路径
```

**区域锚点数据格式**（来自开源项目，树形结构）：

```js
{
  id: '297', name: '安饶之野',
  l_x: -11492, l_y: 4555,      // 左上角坐标
  r_x: -10214, r_y: 5913,      // 右下角坐标（矩形边界）
  parent_id: '0', map_id: '2',
  children: [ /* 子区域，同结构 */ ]
}
```

> 注：开源项目的锚点用**矩形边界**（l_x/l_y/r_x/r_y），本项目需求是**多边形边界**，需在此基础上扩展为 GeoJSON Polygon 坐标数组。

### 3.2 瓦片获取方式

| 方式 | 说明 | 难度 |
|------|------|------|
| **空荧酒馆瓦片抓取** | yuanshen.site 的瓦片可按 `{z}/{x}/{y}` 规律批量下载（需分析其图源URL） | 中 |
| **米哈游官方地图瓦片** | act.mihoyo.com 互动地图的瓦片资源，需抓包分析图源URL | 中 |
| **开源项目自带瓦片** | genshin-map 项目仓库内可能包含部分瓦片资源 | 低 |
| **瓦片下载工具** | 如 MapDownload（开源 electron 工具）支持自定义图源批量下载 | 低 |

### 3.3 分层地图资源（地表/地下/水下）

需求要求地表、地下矿区、海底/水下三层独立图层，对应瓦片来源：

| 图层 | 对应区域 | 瓦片来源 |
|------|---------|---------|
| 地表 | 蒙德/璃月/稻妻/须弥/枫丹/纳塔主地图 | 主地图瓦片 |
| 地下矿区 | 层岩巨渊地下、渊下宫 | 层岩巨渊/渊下宫独立瓦片 |
| 水下 | 枫丹水下区域 | 枫丹水下专属瓦片 |

> 米哈游官方互动地图已内置分层切换（地表/地下/水下），其图源 URL 可通过浏览器开发者工具 Network 面板抓取分析。

## 四、技术参考总结

基于开源项目源码分析，本项目推荐技术落地方案：

### 4.1 地图核心配置（Leaflet）

```js
const map = L.map('map', {
  crs: L.CRS.Simple,              // 游戏地图用简单坐标系
  center: [-84, 148],
  zoom: 5,
  minZoom: 1,
  maxZoom: 7,
  maxBounds: L.latLngBounds([[-77, 56], [-180, 202]]),
  zoomControl: false,
  attributionControl: false
});

// 本地瓦片图层
L.tileLayer('/images/map/{z}/{x}/{y}.png', { maxZoom: 7 }).addTo(map);
```

### 4.2 区域多边形数据结构（JSON 配置文件）

```json
{
  "regions": [
    {
      "id": "mondstadt_city",
      "name": "蒙德城",
      "nation": "蒙德",
      "layer": "surface",
      "polygon": [[lat1,lng1], [lat2,lng2], ...],
      "bgm": {
        "dawn":    { "title": "...", "file": "mondstadt_dawn.mp3", "cover": "..." },
        "day":     { "title": "The City Favored by Wind", "file": "mondstadt_day.mp3", "cover": "..." },
        "dusk":    { "title": "Dusk in Mondstadt", "file": "mondstadt_dusk.mp3", "cover": "..." },
        "night":   { "title": "Mondstadt Starlit", "file": "mondstadt_night.mp3", "cover": "..." }
      }
    }
  ]
}
```

### 4.3 关键依赖库

| 库 | 用途 | 备注 |
|----|------|------|
| leaflet | 瓦片地图、多边形绘制 | 核心库 |
| Web Audio API | 音频淡入淡出、独占播放 | 原生API，无需第三方库 |
| vue3 + vite | 框架与构建 | 需求指定 |
| LocalStorage | 持久化用户设置 | 原生API |

## 五、开发建议与下一步

### 5.1 资源获取优先级

1. **第一步（瓦片）**：从 yuanshen.site 或官方地图抓取一套提瓦特地表瓦片（z=1~7），放入 `public/images/map/`
2. **第二步（映射数据）**：从 Genshin Impact Wiki 各专辑页抓取 6 个国度的曲目-场景-时段映射，整理为 JSON 配置
3. **第三步（音频）**：先从爱给网下载每国度 4 首代表性 BGM（共约 24 首）跑通流程
4. **第四步（多边形边界）**：参考开源项目锚点数据，结合实际地图手动绘制或获取区域多边形坐标

### 5.2 可立即推进的开发工作

资源抓取可与编码并行。以下工作**不依赖外部资源**即可启动：

- Vite + Vue3 项目脚手架搭建
- Leaflet 地图组件封装（先用占位瓦片）
- 原神游戏时钟 UI 与时段逻辑
- Web Audio API 淡入淡出调度引擎
- 唱片播放器弹窗 UI
- 侧边栏框架与 LocalStorage 持久化

待资源就位后接入 JSON 配置即可全功能运转。
